Photon

Photon is a cryptocurrency using Blake256 with POW/AuxPOW and can be Merge Mined With Blakecoin and Others

Get Started

Photon is a child of Blakecoin with a few changes. The standard block award of Photon is 32,768 BUT as the block chain grows the award will increase. The amount it increases is directly related to the current difficulty and height of the blockchain. Miners will get 32,768 coins plus the square root of blockchain height multiplied by the current difficulty.

About

As Photon's are plentiful in the universe the max money is set to 90,000,000,000 That is 90 Billion Photons. Difficulty retargets every 20 blocks with a target of a new block to be produced every 3 minutes. Up to block 3500 the max adjustment is 15% up each retarget. After block 3500 the max adjustment is 3% up each retarget.
